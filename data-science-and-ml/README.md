# Data Science & Machine Learning Journey 🚀

Bu **Repository**, **Software Architecture** ve **Offensive Security** yeteneklerimi **AI** ve **Machine Learning** ile entegre etme yolculuğumu barındırmaktadır. 

Temel amacım; devasa verileri analiz edebilen, **Anomaly Detection** yapabilen ve **Security Tool**'larına entegre edilebilen analitik **Machine Learning Model**'leri geliştirmektir.

## 📁 Directory Structure

* **`00-Basic-Python-OOP-SQL/`** : Python temelleri, **OOP** ve **Database** (**SQL**) pratikleri.
* **`01-Statistics-and-Math/`** : **Data Science**'ın temeli olan istatistik, olasılık ve **Numpy** ile matematiksel hesaplamalar.
* **`02-Data-Analysis-and-EDA/`** : **Pandas**, **Matplotlib** ve **Seaborn** ile **Exploratory Data Analysis (EDA)** ve **Data Visualization**.
* **`03-Feature-Engineering/`** : Ham verileri **Machine Learning Model**'lerine uygun hale getirmek için **Feature Engineering** ve **Data Preprocessing** işlemleri.
* **`04-Machine-Learning-Models/`** : **Regression**, **Classification**, **SVM**, **KNN**, **Random Forest** ve **Anomaly Detection** gibi **Supervised / Unsupervised Learning** algoritmaları.
* **`05-Model-Deployment/`** : Eğitilen modellerin **Export** edilmesi ve **Production** ortamına taşınarak gerçek dünya uygulamalarına dönüştürülmesi.

## 🛠️ Tech Stack & Tools
* **Language:** Python
* **Data Manipulation:** Pandas, Numpy
* **Data Visualization:** Matplotlib, Seaborn
* **Machine Learning:** Scikit-Learn, XGBoost, LightGBM
