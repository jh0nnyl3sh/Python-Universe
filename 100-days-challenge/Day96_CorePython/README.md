# 🗂️ Day 69: Advanced Data Structures & Memory Allocation

> **⚠️ EDUCATIONAL PURPOSE ONLY:**
> This project demonstrates how to effectively group and store dynamically parsed data in `Memory` using nested `Data Structures` before exporting or further processing.

### 🧠 Core Concepts Demonstrated
* **Nested Data Structures:** Combining a `Dictionary` with `List` objects to create a structured temporary database in `Memory`.
* **Dynamic Key Mapping:** Using the output of a parsing function directly as a `Key` to route data into the correct `List` without redundant conditional statements.
* **Data Iteration:** Using the `.items()` method to efficiently iterate over `Dictionary` elements and format the output for the `Console`.

### 📂 File Structure
* `main.py`: The main script handling the `File I/O`, parsing, and data structuring.
* `raw_data/`: A dummy directory containing `.txt` files to simulate raw data drops.

### 🚀 Usage

Run the script directly via terminal:
```bash
python main.py
