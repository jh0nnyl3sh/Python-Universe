# 🔗 Day 91: Linked List - Append Method & Traversal

> **⚠️ EDUCATIONAL PURPOSE ONLY:**
> This project builds upon the `Node` architecture by introducing a `LinkedList` manager Class capable of handling dynamic data insertion.

### 🧠 Core Concepts Demonstrated
* **LinkedList Class Formulation:** Establishing a management structure that tracks the `Head` of the sequence.
* **Dynamic Node Insertion (Append):** Writing an algorithm to add new `Node` objects to the end of the list.
* **Pointer Traversal:** Using a `while` loop to navigate through memory by following `next` references until the end of the chain is reached.
* **Edge Case Handling:** Safely managing scenarios where the `Linked List` is entirely empty before insertion.

### 📂 File Structure
* `main.py`: Contains both the `Node` and `LinkedList` classes, along with the execution logic to test the `append` method.
