# 🔗 Day 90: Linked List Architecture - Core Nodes

> **⚠️ EDUCATIONAL PURPOSE ONLY:**
> This project demonstrates the absolute fundamentals of the `Linked List` data structure, specifically focusing on `Class` creation and `Pointer` manipulation in memory.

Unlike standard `List` arrays where memory blocks are contiguous, a `Linked List` relies on independent `Node` objects scattered in memory, chained together via pointers. 

### 🧠 Core Concepts Demonstrated
* **Node Class Creation:** Building the fundamental blueprint that holds `value` and `next` attributes.
* **Object Instantiation:** Generating independent `Node` objects in memory.
* **Pointer Linking:** Manually manipulating the `next` reference of one node to point to the memory address of another, establishing the primary chain of a `Linked List`.

### 📂 File Structure
* `main.py`: Contains the `Node` class definition and the execution logic to link two nodes together and traverse them.
