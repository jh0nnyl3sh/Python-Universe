## Overview
This project marks the official entry into Object-Oriented Programming (OOP) in Python. It demonstrates the core principles of creating a `Class` template and instantiating multiple independent `Objects` from it.

## Key Concepts Mastered
* **Classes & Objects:** Built a `GhostOperative` blueprint and deployed distinct agent instances.
* **Constructors (`__init__`):** Used initialization methods to assign unique attributes (codename, stealth_level) to each object upon creation.
* **Instance Methods:** Developed custom capabilities for the objects, utilizing the `self` keyword to access instance-specific data.
* **Environment Security:** Successfully configured Python Virtual Environment (`venv`) and `.gitignore` to maintain a clean and secure repository.

Run the script to see the agents report their live status:
