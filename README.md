# 🐍 Python Universe

Bu depo, Python ekosisteminde geliştirdiğim otomasyon araçlarını, web scraping projelerini ve veri yönetimi çalışmalarını içeren kapsamlı bir koleksiyondur. 

## 🏗️ Mimari Yapı

- **📂 automation-tools:** Günlük rutinleri hızlandıran mesaj gönderici ve iş akışı araçları.
- **📂 web-scraping:** Adalet Bakanlığı duyuruları ve veri madenciliği odaklı dinamik tarayıcılar.
- **📂 database-tools:** PostgreSQL ve SQL tabanlı veritabanı yönetim betikleri.
- **📂 projects:** Envanter yönetim sistemleri ve arıza takip gibi uçtan uca çözümler.
- **📂 practice-archive:** Python temelleri ve algoritma çalışmaları.

## 🚀 Öne Çıkan Özellikler
- Modüler kod yapısı ve temiz mimari.
- Gerçek dünya problemlerine (kamu verisi işleme vb.) yönelik çözümler.
- GitHub geçmişi (history) korunarak taşınmış subtree yapısı.
