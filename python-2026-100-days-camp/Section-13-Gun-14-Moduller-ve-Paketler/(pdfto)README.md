# PDF to DOCX Converter

Bu **script**, `pdf2docx` **library**'sini kullanarak bir PDF dosyasını Word belgesi (`.docx`) formatına **convert** eder. İçerisinde **exception handling** barındırır ve Python geliştirme için temel **best practice** yönergelerine uyar.

## Ön Koşullar

Bu **script**'i çalıştırmadan önce, sisteminizde gerekli **dependency**'lerin kurulu olduğundan emin olun.

```bash
pip install pdf2docx
